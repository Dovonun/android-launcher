# Development

## Build
Use the Gradle wrapper from the repository root.

```sh
./gradlew :app:compileDebugKotlin
```

## Tests
```sh
./gradlew test
```

## Local environment
Machine-specific Java configuration belongs in the developer's shell or IDE setup, not in repo docs.
