# Ni-Launcher

Ni-Launcher is a fast, minimal Android launcher focused on stability and speed. It avoids layout shift and keeps navigation quick and predictable so you can move through apps without visual jitter or waiting.

**Values**
- No layout shift
- Fast navigation

**Highlights**
- Nested tags instead of folders
- Quick app access with tag-driven organization
- Minimal UI with gesture-first flows

## Installation

### APK
1. Navigate to the [Releases](https://github.com/Dovonun/ni-launcher/releases) page.
2. Download the latest `ni-launcher-release.apk`.
3. Open the file on your device to install.

### Obtainium
For automated updates via [Obtainium](https://github.com/ImranR98/Obtainium), add this repository URL:
`https://github.com/Dovonun/ni-launcher`
