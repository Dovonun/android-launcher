# Technology Stack

## Core Technologies
- **Language:** [Kotlin](https://kotlinlang.org/) (2.2.20) - Selected for modern, concise, and safe Android development.
- **Platform:** Android SDK (API 35+) - Targeting the latest Android features and security standards.
- **UI Framework:** [Jetpack Compose](https://developer.android.com/compose) - A modern toolkit for building native UI with a reactive, declarative approach.
- **UI Components:** [Material Design 3](https://m3.material.io/) - Implementing the latest Android design standards for a polished, native feel.

## Data & Persistence
- **Database:** [Room](https://developer.android.com/training/data-storage/room) (2.8.1) - A robust abstraction layer over SQLite for local data persistence (favorites, settings).
- **Architecture:** MVVM (Model-View-ViewModel) - Utilizing Android Lifecycle ViewModels to separate UI logic from business logic.

## Build & Tooling
- **Build System:** Gradle Kotlin DSL - Providing a typesafe and scriptable build configuration.
- **Dependency Management:** Gradle Version Catalog (libs.versions.toml) - A centralized and scalable way to manage project dependencies.
